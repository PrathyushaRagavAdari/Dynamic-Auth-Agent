# System Setup Instructions
1. **Clone the repository:** `git clone https://github.com/PrathyushaRagavAdari/Dynamic-Auth-Agent.git`
2. **Navigate to directory:** `cd Dynamic-Auth-Agent`
3. **Environment Setup:** Ensure you have Python 3.10+ installed.
4. **Environment Variables:** Create a `.env` file in the root directory and add:
   `OPENAI_API_KEY=your_key_here`
   `SNOWFLAKE_USER=your_user`
